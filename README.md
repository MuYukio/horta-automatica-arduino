#  Horta Automática com Arduino

Projeto de **sistema de irrigação automatizado utilizando Arduino**, desenvolvido com foco em **sustentabilidade**, **automação** e **aplicação educacional**, especialmente voltado para o contexto de **hortas escolares**.

O projeto automatiza o processo de irrigação a partir da leitura da umidade do solo, contribuindo para o uso consciente da água e para o ensino prático de tecnologia, eletrônica e programação.

---

##  Visão Geral

A Horta Automática utiliza sensores de umidade do solo para monitorar as condições das plantas e acionar automaticamente o sistema de irrigação quando necessário.  
O controle é realizado por um microcontrolador Arduino, responsável pela tomada de decisão, registro de eventos e acionamento dos atuadores.

Além do aspecto técnico, o projeto possui forte **caráter educacional e sustentável**, sendo aplicado em ambiente escolar e utilizado como ferramenta de aprendizagem interdisciplinar.

---

##  Funcionalidades

- Mede automaticamente a umidade do solo
- Decide quando irrigar com base nos dados coletados
- Liga e desliga a irrigação sem intervenção humana
- Registra o histórico de funcionamento do sistema
- Exibe informações do sistema em tempo real
- Funciona de forma autônoma e segura
- Pode ser aplicado em hortas e viveiros escolares

##  Componentes Utilizados

- Arduino (Uno, Mega ou compatível)
- Sensores de umidade do solo
- Módulo relé
- Válvulas solenoide e/ou bomba de água
- Display LCD (opcional)
- Fonte de alimentação / bateria
- Jumpers, protoboard e demais componentes eletrônicos

##  Como utilizar o projeto

1. Abra o arquivo `.ino` na **Arduino IDE**
2. Conecte a placa Arduino ao computador
3. Selecione a placa e a porta corretas
4. Faça o upload do código
5. Monte o circuito conforme descrito no manual técnico
6. Alimente o sistema e acompanhe o funcionamento da irrigação automática
